# YouTube Channel Search App

This app enables users to search videos from "LearnWithNaw" channel. It uses YouTube API. This app will NOT work, unless you provide your YouTube API key in `src/api/key.js`

## Technologies used: 
- React/Redux 
- HTML
- CSS (flexbox / grid)
